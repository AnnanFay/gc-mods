  • Moved json unparsing to getGlobalValue
  • Automated ship data harvesting
  • Removed usused consoles
  • Added verification of auth token
  • Status change support
  • Loading gif added to css
  • Better loading css
  • Set authentication flags on right on first auth success
  • Loading animation support added to correct xhrs
  • Changed with of auth token input field to accommodate the loading gif
  • Fixed a typo in get_statuses API

  • Moved status-handling to accommodate new server-side API and added a couple of fixes to that
